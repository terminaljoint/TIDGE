TECHNICAL SPECIFICATIONS
================================================================================

MATH SYSTEM:
  - Vec3: 3D vectors with operations (add, sub, mul, dot, cross, norm, len)
  - Matrix4: 4x4 transformation matrices with composition

ANIMATION:
  - Keyframe: Individual animation point
  - AnimationCurve: Collection of keyframes with interpolation
  - AnimationClip: Complete animation (9 curves)
  - AnimationController: Manages and updates animations

GRAPHICS:
  - Transform: Position + Rotation + Scale with matrix math
  - Geometry: Procedural generation (5 types)
  - Material: PBR properties (color, metalness, roughness)
  - Renderer: Canvas 2D with grid drawing and entity rendering

SCENE:
  - Entity: Game object with mesh, transform, animation
  - Scene: Entity container with hierarchy
  - GameLoop: Main update loop with delta time (60 FPS target)
